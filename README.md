# DocStable – Official Downloads

This repository contains the official release builds of **DocStable**.

DocStable is a deterministic, offline decision-support tool for Microsoft Office document migration and risk analysis.

## Download

The latest stable version can be found under:

➡️ **Releases**

Each release includes:

- Windows installer (.exe)
- Release notes
- SHA-256 checksums (where applicable)

## About DocStable

DocStable is designed for organizations and public institutions that require:

- Transparent risk assessment
- Deterministic analysis (no cloud, no AI black box)
- Fully local processing
- Clear, explainable findings

For more information, please visit the official project website.
